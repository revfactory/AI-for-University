# 교재 집필 가이드

## 문체 원칙

1. **대화체 + 전문성** — 친근하되 가볍지 않게. "~합니다" 체를 기본으로, 팁이나 주의사항은 "~해요" 체 혼용 가능
2. **실용 우선** — 이론은 최소한으로, "왜 이게 필요한지" → "어떻게 하는지" 순서
3. **구체적 예시** — 추상적 설명보다 실제 프롬프트, 실제 결과물, 실제 스크린샷
4. **Before/After** — 가능한 모든 곳에서 AI 사용 전후를 비교
5. **도구 중립** — 특정 도구를 편향되게 추천하지 않되, 상황별 최적 도구는 명시

## 프롬프트 예시 작성법

프롬프트 예시는 교재의 핵심 자산이다. 다음 규칙을 따른다:

```
💬 프롬프트 예시

[목적]: 경영학 레포트 개요 작성
[도구]: ChatGPT / Claude

---
나는 경영학과 3학년 학생입니다. "디지털 트랜스포메이션이 중소기업 경쟁력에 미치는 영향"이라는
주제로 A4 10장 분량의 레포트를 작성해야 합니다.

다음 조건에 맞는 레포트 개요를 작성해주세요:
1. 서론-본론-결론 구조
2. 본론은 3~4개 소주제로 구성
3. 각 소주제마다 다룰 핵심 논점 2~3개
4. 참고할 만한 이론이나 프레임워크 제안
5. 한국 중소기업 사례를 포함할 수 있는 부분 표시
---

📌 이 프롬프트가 좋은 이유:
- 본인의 맥락(학과, 학년)을 제공
- 구체적인 조건(분량, 구조)을 명시
- 기대하는 출력 형식을 구조화
```

## 실전 사례 작성법

실증 사례는 반드시 다음 정보를 포함:
- **누가**: 어떤 학교/학과/학년의 학생(또는 교수)
- **무엇을**: 어떤 작업에 AI를 활용
- **어떻게**: 구체적 도구와 방법
- **결과**: 측정 가능한 성과 (시간 절약, 성적 향상, 품질 개선 등)
- **출처**: 기사, 논문, 블로그, 인터뷰 등

사례가 확인되지 않은 경우, 가상 사례임을 명시하되 현실적으로 작성한다.

## Claude Code 섹션 작성법

각 챕터의 CC 섹션은 다음 구조를 따른다:

```markdown
### 🔧 Claude Code & 하네스 활용

#### 기본 활용
- 이 챕터의 작업을 Claude Code로 수행하는 방법
- 실제 명령어 및 대화 예시

#### 자동화 레시피
- 반복 작업을 자동화하는 구체적 워크플로우
- 스킬/에이전트 활용 예시 (해당 시)

#### 하네스 심화 (선택)
- 에이전트 팀 구성 예시
- 대규모 작업 자동화 사례
```

## 분량 가이드

| 섹션 | 분량 (A4 기준) | 비고 |
|------|--------------|------|
| 이런 분이 읽으면 좋아요 | 0.5장 | |
| 학습목표 | 0.5장 | 완성 결과물 미리보기 포함 |
| AI 도구 소개 + 비교 실험 | 2~3장 | 도구 비교 실험 필수 포함 |
| 핵심 개념 + 프롬프트 진화 | 2~3장 | 🔴→🟡→🟢 패턴 필수 포함 |
| 따라하기 실습 | 4~6장 | **교재의 핵심**. Step-by-step 완전 가이드 |
| 실전 사례 | 1~2장 | A/B급 사례 중심, Before/After 수치 |
| 꿀팁 & 주의사항 | 1~2장 | Anti-Pattern + 워크플로우 일반화 필수 |
| 도전 과제 | 1장 | 3단계 (⭐/⭐⭐/⭐⭐⭐) |
| CC 섹션 | 2~3장 | 실행 가능한 코드 필수 |
| **챕터 합계** | **약 15~20장** | 이미지 5~7장 포함 |

## 참고문헌 및 출처

- 모든 외부 정보에는 출처를 명시
- URL은 접근 가능한 것만 포함
- 논문은 저자(연도) 형식, 웹은 "사이트명, 제목, URL, 접근일" 형식
