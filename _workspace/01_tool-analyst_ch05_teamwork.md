### 1.3 클로바노트 상세

**개인용 (무료):**
- 음성 녹음 → 텍스트 변환 → 요약
- 한국어, 영어, 일본어, 중국어 인식
- 주요내용, 다음 할 일, 단락별 내용 자동 정리

**비즈니스용 (네이버웍스):**
- 관리·보안·감사·협업 기능 추가
- Lite: ₩20,000/월 (연간 계약 시 ₩18,000/월)
- 팀 워크스페이스, 권한 관리

**학생 활용 팁:**
- 팀 회의 시 스마트폰으로 녹음 → 클로바노트로 자동 전사
- 요약 결과를 ChatGPT에 붙여넣고 "이 회의 내용을 기반으로 회의록 양식에 맞춰 정리해줘" 요청
- 회의 → 클로바노트 → ChatGPT/Claude → 최종 회의록 파이프라인

## 2. 역할 분담·일정 관리 AI 도구

### 2.1 Notion AI 심층 분석

**핵심 기능:**
- 올인원 워크스페이스: 노트, 데이터베이스, 위키, 프로젝트 관리
- **Ask Notion:** 워크스페이스 전체에서 자연어 질문 → 페이지/위키/DB에서 답변 (출처 링크 포함)
- **AI Agent (Notion 3.0):** 멀티스텝 워크플로우를 자율 실행하는 에이전트
- **다중 AI 모델:** GPT-5, Claude Opus 4.1, o3 등 프리미엄 모델 지원

**가격:**
| 플랜 | 가격 | AI 접근 |
|------|------|---------|
| Free | $0 | AI 체험 20회 |
| Plus | $10/유저/월 (연간) | AI 체험 20회 |
| Business | $20/유저/월 | **AI 전체 접근** |
| Enterprise | 맞춤 | AI 전체 + 고급 보안 |

**주의:** Free/Plus는 AI 체험 20회만 제공. 실질적 AI 활용에는 Business 플랜 필수.

**팀플 활용 워크플로우:**
```
1. Notion에 팀 프로젝트 페이지 생성
2. AI에게 "이 프로젝트를 4명이 4주간 진행할 때 역할 분담과 일정을 제안해줘" 요청
3. 칸반보드/타임라인 뷰로 진행 상황 관리
4. 각자 작성한 내용을 Ask Notion으로 통합 검색
```

### 2.2 ChatGPT/Claude를 활용한 역할 분담

**프롬프트 예시:**
```
우리 팀은 4명으로 구성되어 있고, "AI 시대의 미디어 리터러시" 주제로
발표를 준비해야 합니다. 발표일은 2주 후입니다.

팀원 정보:
- A: 자료조사에 강함, 발표 경험 적음
- B: PPT 디자인에 능함
- C: 발표/스피치에 자신 있음
- D: 데이터 분석/통계 가능

다음을 제안해주세요:
1. 역할 분담 (각자의 강점 활용)
2. 2주간 주차별 일정표 (마일스톤 포함)
3. 각 역할에서 활용할 수 있는 AI 도구 추천
4. 중간 점검 체크리스트
```

## 3. 팀 작업 통합 및 일관성 유지

### 3.1 AI로 통합하는 워크플로우

```
┌────────────────────────────────────────────────────┐
│ Phase 1: 기획 (AI 활용)                             │
│ → ChatGPT/Claude에게 프로젝트 구조/역할 분담 요청    │
│ → Notion AI로 프로젝트 페이지 자동 생성              │
├────────────────────────────────────────────────────┤
│ Phase 2: 개별 작업                                  │
│ → 각자 담당 파트를 AI 도구로 작업                    │
│ → 공유 문서(Google Docs/Notion)에 작성               │
├────────────────────────────────────────────────────┤
│ Phase 3: 통합 (AI 핵심 활용 포인트!)                 │
│ → Claude에 전체 문서 업로드                          │
│ → "4명이 작성한 이 보고서의 문체/톤/용어를            │
│    통일해주세요. 일관된 학술적 문체로 교정해주세요"   │
│ → Gemini Docs에서 직접 통합 편집                     │
├────────────────────────────────────────────────────┤
│ Phase 4: 검토 & 피드백                              │
│ → Claude: "이 보고서에서 논리적으로 약한 부분은?"     │
│ → ChatGPT: "교수님 관점에서 이 보고서를 평가해줘"    │
│ → Perplexity: 핵심 주장의 출처 검증                  │
└────────────────────────────────────────────────────┘
```

### 3.2 AI로 피드백 주고받기

**팀원 간 상호 피드백 프롬프트:**
```
다음은 팀원이 작성한 보고서 섹션입니다.
건설적이고 구체적인 피드백을 3가지 제안해주세요.

피드백 형식:
1. 잘한 점 (구체적으로)
2. 개선이 필요한 점 (대안 포함)
3. 추가하면 좋을 내용

[팀원 작성 내용 붙여넣기]
```

## 4. 상황별 최적 도구 선택 가이드

| 팀플 단계 | 추천 도구 | 이유 |
|----------|----------|------|
| 주제 브레인스토밍 | ChatGPT | 다양한 아이디어 빠르게 생성 |
| 자료 조사 | Perplexity + Semantic Scholar | 출처 포함 검색, 무료 학술 DB |
| 역할 분담/일정 | Notion AI or ChatGPT | 팀원 특성 기반 분배 |
| 회의록 작성 | 클로바노트 + ChatGPT | 한국어 전사 → AI 정리 |
| 개별 파트 작성 | Claude/ChatGPT | 각자 강점에 맞는 도구 |
| 문서 통합 | Claude (1M 컨텍스트) | 전체 문서를 한번에 분석/통일 |
| 발표자료 제작 | Gamma/Canva/Google Slides+Gemini | AI 슬라이드 생성 |
| 최종 검토 | Perplexity + Claude | 팩트체킹 + 논리 검토 |
