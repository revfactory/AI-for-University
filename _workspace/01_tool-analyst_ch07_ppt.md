### 1.2 Gamma 심층 분석

**핵심 기능 (2025~2026):**
- **Gamma Agent (3.0, 2025.09):** 자연어 대화로 리서치, 콘텐츠 수정, 리스타일링, 복잡한 편집 실행
- **다중 출력:** PPT, PDF, PNG, Google Slides 내보내기. 웹사이트로 퍼블리시 가능 (커스텀 도메인)
- **AI Animations:** Ultra/Business 플랜에서 자동 애니메이션 (2026.01)
- **카드 기반 포맷:** 전통적 슬라이드가 아닌 스크롤 가능한 웹 카드 형식

**가격 상세:**
| 플랜 | 가격 | AI 크레딧 | 주요 기능 |
|------|------|----------|----------|
| Free | $0 | 400 | 기본 생성, Gamma 워터마크 |
| Plus | $8/월 (연간) | 무제한 | 워터마크 제거, 커스텀 폰트 |
| Pro | $15/월 (연간) | 무제한 | 분석, 커스텀 도메인 |
| Ultra | $90/월 (연간) | 무제한 | AI Animations, 우선 지원 |

**적합한 상황:** 빠른 초안, 웹 공유, 아이디어 발표
**부적합한 상황:** 공식 PPT 발표 (내보내기 레이아웃 깨짐), 기업 브랜딩

### 1.3 Beautiful.ai 심층 분석

- **Smart Slides:** 콘텐츠에 따라 레이아웃이 자동 조정
- **브랜드 통제:** 색상, 폰트, 로고를 팀 전체에 일관 적용
- **PPT 내보내기:** 업계 최고 수준의 PowerPoint 호환성
- **가격:** Pro $12/월 (연간), Team $40/유저/월
- **약점:** 무료 플랜 없음, 비영어권 지원 약함

## 2. 발표 스크립트·Q&A 자동생성

### 2.1 도구별 스크립트 생성 비교

| 도구 | 스크립트 품질 | Q&A 생성 | 적합한 상황 |
|------|-------------|---------|-----------|
| **ChatGPT** | ★★★★★ | ★★★★★ | 슬라이드별 스크립트 + 예상 질문 + 답변 전략 생성에 최적 |
| **Claude** | ★★★★★ | ★★★★☆ | 학술 발표 스크립트, 논리적 흐름 강조 |
| **Gemini** | ★★★★☆ | ★★★★☆ | Google Slides 연동, 슬라이드 노트 직접 생성 |
| **Gamma** | ★★★☆☆ | ★★☆☆☆ | 기본 발표자 노트 자동 생성 |

### 2.2 발표 스크립트 S급 프롬프트

```
다음은 내 발표 슬라이드의 내용입니다:
[슬라이드 내용 붙여넣기]

이 발표에 대해 다음을 작성해주세요:

1. 슬라이드별 발표 스크립트 (각 1~2분 분량, 구어체)
2. 청중 시선을 잡는 오프닝 멘트
3. 슬라이드 전환 연결 문구 ("이어서~", "다음으로~")
4. 예상 질문 5개 + 답변 전략 (STAR 기법 적용)
5. 클로징 멘트 (핵심 메시지 3줄 요약)

발표 상황:
- 대상: 경영학 수업 학생 30명 + 교수님
- 시간: 15분 발표 + 5분 Q&A
- 분위기: 학술적이되 지루하지 않게
```

## 3. 디자인 원칙과 AI

### 3.1 AI 프레젠테이션 디자인 체크리스트

```
✅ 한 슬라이드에 하나의 핵심 메시지만
✅ 텍스트 최소화 (6x6 규칙: 6줄, 줄당 6단어 이하)
✅ 고대비 색상 조합 (AI에게 "접근성 높은 색상 팔레트 제안해줘")
✅ 일관된 폰트 (제목 1종 + 본문 1종)
✅ 여백 충분히 확보
❌ 워드아트, 클립아트, 과도한 애니메이션 지양
❌ AI 기본 템플릿 그대로 사용하지 않기 (커스터마이징 필수)
```

### 3.2 상황별 최적 도구 선택

| 상황 | 추천 도구 | 이유 |
|------|----------|------|
| 수업 발표 (빠른 제작) | Gamma Free | 무료, 60초 생성, 웹 공유 |
| 팀 발표 (PPT 제출) | Canva → PPT 내보내기 | 팀 공동 편집 + PPT 호환 양호 |
| 학술 발표 (학회) | Beautiful.ai 또는 ChatGPT+PPT 수동 | 깔끔한 디자인, 정확한 내보내기 |
| 졸업 발표/포트폴리오 | Gamma Pro (웹) 또는 Canva Pro | 전문적 디자인, 인터랙티브 |

## 4. 프롬프트 진화 패턴 (PPT 제작용)

### 🔴 Level 1
```
AI 관련 발표자료 만들어줘
```
→ 결과: 너무 일반적, 디자인 지시 없음

### 🟡 Level 2
```
"대학생의 AI 활용 현황" 주제로 10슬라이드 발표자료를 만들어줘.
```
→ 결과: 구조는 있지만 청중/시간/스타일 맥락 부족

### 🟢 Level 3 (S급)
```
Gamma 또는 ChatGPT에서:

주제: "대학생의 AI 활용 현황과 미래"
목적: 교양수업 기말 팀 발표
청중: 대학생 40명 + 교수님
시간: 15분

슬라이드 구성:
1. 표지 (주제, 팀명, 날짜)
2. 목차
3. AI 도구 사용 현황 (통계 차트)
4-6. 주요 활용 사례 3가지 (각 슬라이드에 사례 1개)
7. 문제점과 윤리적 이슈
8. 해결 방안 제안
9. 결론 및 시사점
10. Q&A / 참고문헌

디자인 요청:
- 색상: 파란색 계열 + 화이트 배경 (학술적)
- 각 슬라이드에 관련 아이콘 또는 인포그래픽 포함
- 텍스트 최소화, 키워드 + 시각자료 중심
```
