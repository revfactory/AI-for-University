## 1. AI 직무 탐색·경력 설계 도구 비교

### 1.1 주요 도구 비교표

| 도구 | 가격 | 핵심 기능 | 한국어 | 적합한 상황 |
|------|------|----------|--------|-----------|
| **ChatGPT** | Free / Plus $20/월 | 직무 분석, 역량 매칭, 경력 로드맵, 채용공고 분석 | ★★★★★ | **1순위 — 가장 범용적** |
| **Claude** | Free / Pro $20/월 | 심층 직무 비교, 산업 구조 분석, 장문 경력 설계 | ★★★★★ | 복잡한 경력 설계 |
| **Perplexity** | Free / Pro $20/월 | 실시간 채용 시장 분석, 연봉 데이터, 기업 비교 | ★★★☆☆ | 시장 조사, 트렌드 파악 |
| **Google Career Dreamer** | **무료** | AI 경력 탐색, 스킬 매칭, 경력 정체성 생성 | ★★☆☆☆ | 진로 탐색 초기 단계 |
| **CareerExplorer** | Free 기본 / 유료 | ML 기반 커리어 추천, 만족도 예측, 성장성 분석 | ★☆☆☆☆ | 해외 커리어 탐색 |
| **Teal** | Free / $7~19/월 | 잡 트래킹 + 이력서 + 스킬 매칭 통합 | ★☆☆☆☆ | 해외 취업 관리 |

### 1.2 Google Career Dreamer 심층 분석

**출시:** 2025.12 (Grow with Google 프로그램)
**가격:** 무료

**핵심 기능:**
1. **경력 정체성 문구(Career Identity Statement) 생성:**
   - 역할, 스킬, 경험, 학력, 관심사를 선택 → AI가 커리어 정체성 문구 작성
   - 이력서·면접 토킹포인트로 활용 가능

2. **시각적 경력 탐색:**
   - 비주얼 웹 형태로 적합한 커리어 경로 표시
   - 관심 있는 직무를 클릭하면 상세 정보 확인
   - Lightcast + 미국 노동통계국(BLS) 취업 시장 데이터 기반

3. **자격증/교육 연결:**
   - 선택한 커리어에 필요한 자격증·인증 표시
   - Google Cloud Skills Boost, Google Career Certificates 연결

4. **Gemini 연동:**
   - 커버레터·이력서 작성 협업
   - 추가 직업 아이디어 탐색

**한계:** 영어 전용, 미국 취업 시장 데이터 중심 → 한국 학생은 참고용

### 1.3 CareerExplorer (by Sokanu) 분석

- **ML 기반 커리어 추천:** 퀴즈/설문 → 관심사·적성 분석 → 커리어 매칭
- **데이터 기반:** 커리어 만족도 트렌드, 산업 성장성, 실제 취업 데이터
- **이전 가능한 스킬(Transferable Skills) 분석:** 전공에서 어떤 직무로 이동 가능한지 매핑
- **한계:** 영어, 해외 시장 중심

## 2. 채용공고 분석 AI 활용

### 2.1 채용공고 분석 프롬프트

```
다음 채용공고를 분석해주세요:

[채용공고 전문 붙여넣기]

분석 내용:
1. 핵심 자격 요건 (필수 vs 우대)
2. 요구 역량 리스트 (하드스킬 + 소프트스킬)
3. JD(직무기술서)에서 읽을 수 있는 숨은 의미
4. 이 직무의 일상적인 업무 내용 추정
5. 나의 역량과의 매칭도 분석:
   - 나의 역량: [내 스킬/경험 나열]
   - 매칭 ✅ / 부족 ⚠️ / 미보유 ❌ 표시
6. 부족한 역량을 보완하기 위한 구체적 액션 플랜
7. 이 공고에 맞춘 자소서 키워드 5가지
```

### 2.2 채용공고 비교 분석

```
다음 3개 채용공고를 비교 분석해주세요:

[공고 1 붙여넣기]
[공고 2 붙여넣기]
[공고 3 붙여넣기]

비교 항목:
1. 요구 역량 매트릭스 (표 형태)
2. 연봉/복지 비교 (알 수 있는 범위에서)
3. 성장 가능성 (기업 규모, 산업 전망)
4. 나의 적합도 순위 (1위~3위)
5. 각 공고별 합격 전략 차이점
```

## 3. 역량 매칭 + 경력 로드맵

### 3.1 역량 갭 분석 프롬프트

```
나는 [전공] [학년] 학생이고, [희망 직무]에 취업하고 싶습니다.

현재 나의 역량:
- 학력: [대학교, 전공, 학점]
- 자격증: [보유 자격증]
- 경험: [인턴/프로젝트/동아리]
- 기술: [보유 기술/도구]
- 외국어: [어학 성적]

분석해주세요:
1. [희망 직무]의 핵심 역량 리스트 (10가지)
2. 나의 현재 역량 매칭도 (각 역량별 ✅/⚠️/❌)
3. 역량 갭(Gap) 분석표
4. 부족한 역량별 구체적 보완 방법
5. D-6개월 / D-3개월 / D-1개월 준비 로드맵
6. 동일 직무 합격자의 평균 스펙 (예상)
```

### 3.2 경력 로드맵 생성 프롬프트

```
[희망 직무]의 5년 / 10년 경력 로드맵을 설계해주세요.

현재 상태: [대학교 N학년, 전공]
희망 진로: [1차 목표 직무] → [장기 목표]

포함 내용:
1. 연차별 포지션 변화 (신입 → 3년차 → 5년차 → 10년차)
2. 각 단계에서 필요한 역량/자격증
3. 연봉 범위 예상 (한국 시장 기준)
4. 커리어 분기점 (어떤 선택지가 있는지)
5. 현재부터 준비해야 할 것 (우선순위 Top 5)
6. 추천 자격증/교육 프로그램
```

### 3.3 산업 분석 프롬프트

```
[산업명] 산업에 대해 분석해주세요:

1. 산업 구조 (주요 기업, 밸류체인)
2. 시장 규모 및 성장 전망 (한국 기준)
3. 핵심 직무와 필요 역량
4. 연봉 수준 (직무별, 연차별)
5. AI/디지털 전환이 이 산업에 미치는 영향
6. 대학생이 이 산업에 진입하는 최적 경로
7. 이 산업의 채용 트렌드 (2025~2026)

가능하면 최신 데이터와 출처를 포함해줘.
```

## 4. 직무 탐색 워크플로우

### 4.1 전체 파이프라인

```
[Phase 1: 자기 탐색] (Google Career Dreamer / ChatGPT)
  → 나의 관심사, 강점, 가치관 정리
  → Career Identity Statement 생성
  → 잠재적 직무 리스트 도출

[Phase 2: 시장 조사] (Perplexity)
  → 관심 산업 트렌드, 채용 시장, 연봉 데이터
  → 실시간 채용공고 분석

[Phase 3: 역량 매칭] (ChatGPT / Claude)
  → 채용공고 JD 분석 → 역량 갭 파악
  → 부족 역량 보완 계획 수립

[Phase 4: 경력 설계] (ChatGPT / Claude)
  → 5년/10년 로드맵 생성
  → 단기(재학 중) + 중기(취업 후) 목표 설정

[Phase 5: 실행] (Teal / Notion)
  → 지원 기업 트래킹
  → 일정 관리 + 준비 체크리스트
```

### 4.2 프롬프트 진화 패턴

**🔴 Level 1 (초보):**
```
마케팅 직무에 대해 알려줘
```

**🟡 Level 2 (중급):**
```
IT기업 마케팅 직무의 필요 역량과 채용 트렌드를 분석해줘
```

**🟢 Level 3 (고급):**
```
경영학과 4학년, 디지털 마케팅 직무 취업 준비:
■ 현재 역량: 마케팅원론 A+, Google Analytics 자격증, SNS 운영 경험 (인스타 팔로워 5K)
■ 분석 요청:
1. 디지털 마케팅 핵심 역량 10가지 × 내 매칭도
2. 역량 갭 Top 3와 각각의 보완 방법 (구체적 강좌/자격증)
3. 네이버/카카오/쿠팡 마케팅팀 채용 비교
4. 6개월 취업 준비 로드맵 (월별 마일스톤)
5. 포트폴리오에 포함할 프로젝트 제안 3가지
한국 채용 시장 기준, 2026년 트렌드 반영
```
