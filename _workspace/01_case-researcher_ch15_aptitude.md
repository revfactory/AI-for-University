## 7. 실패 사례 / Anti-Pattern

### 사례 7-1: AI로 인적성 시험 "풀이"만 공부하는 함정 (✅ A급 — 교차검증)

**출처:** 사례 1-1, 2-1 교차검증

**핵심 교훈:**
- ChatGPT의 수리/논리 문제 풀이: 기본은 정확하나 **고난도 추론에서 오류** (사례 1-1)
- GSAT은 문제당 30~60초 — **풀이 이해가 아닌 속도 훈련**이 핵심 (사례 2-1)
- AI로 풀이 과정을 이해하는 것은 좋지만, **시간 내 풀기는 반복 연습으로만 가능**

**학생이 주의할 점:**
1. AI 풀이 설명으로 **개념 이해** (O) → AI 풀이에만 의존 (X)
2. 실전은 **타이머 켜고 CBT 모의시험** 반복
3. AI가 만든 문제는 난이도 보정 필요 → **기출 기반으로 검증**

**교재 활용:** "AI로 개념 이해, 실전은 타이머로" — 핵심 Anti-Pattern

---

### 사례 7-2: AI 생성 문제의 난이도 불일치 (💡 C급)

**출처:** 합리적 추론 기반 구성 사례

**시나리오:**
- 대학생 G가 ChatGPT에 "GSAT 수리논리 문제 20개 만들어줘" 요청
- ChatGPT가 생성한 문제: 10개는 너무 쉽고 10개는 너무 어려움
- 실제 GSAT 난이도: 중간 수준이 대부분이나 함정 문제가 포함
- **교훈:** AI 생성 문제는 **실제 시험 난이도와 출제 패턴을 정확히 반영하지 못함**
- 기출 문제 분석이 선행되어야 AI 문제 생성이 의미 있음

**교재 활용:** "AI 문제와 실제 GSAT의 난이도 차이"

### NCS 통합채용 구성
- **5개 영역** 각 10문항, **총 50문항**
- AI로 영역별 맞춤 문제 생성 시 참고

### ChatGPT 인적성 문제 생성 정확도·한계 (tool-analyst 검증)
- **강점:** NCS 수리능력/문제해결능력 문제 생성에 뛰어남. 풀이 과정 설명, 오답 분석, 약점 진단까지 원스톱
- **한계:**
  - 실제 GSAT/NCS 기출과 100% 동일한 난이도 보장 불가 (기출 데이터 학습 제한)
  - 도표/그래프 기반 문제는 텍스트로 표현 → 실제 시험 형태와 다름
  - 수학적 계산 오류 간헐적 발생 → 풀이 과정 직접 검증 필요
- **추천 활용법:** 유형 익히기 + 약점 보완 연습용, 최종 실전은 기출문제집 병행
