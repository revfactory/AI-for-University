## 1. Claude Code의 파일 처리 능력

### 1-1. 지원하는 파일 작업

Claude Code는 **파이썬 스크립트를 자동 생성·실행**하여 다양한 파일을 처리합니다:

| 파일 형식 | 가능한 작업 | 사용 라이브러리 |
|----------|-----------|--------------|
| **CSV** | 읽기, 쓰기, 필터링, 집계, 병합 | pandas (내장) |
| **XLSX** | 읽기, 쓰기, 수식, 서식, 차트 | openpyxl |
| **DOCX** | 생성, 수정, 표 삽입, 서식 | python-docx |
| **JSON** | 읽기, 쓰기, 변환 | json (내장) |
| **TXT** | 읽기, 쓰기, 패턴 추출 | 내장 |

핵심 포인트: Claude Code에게 "이 엑셀 파일을 분석해줘"라고 요청하면, 적절한 파이썬 코드를 자동 생성하고 실행합니다. 프로그래밍 지식이 없어도 됩니다.

### 1-2. 파이썬 라이브러리 자동 설치

필요한 라이브러리가 없으면 Claude Code가 자동으로 설치합니다:

```bash
# Claude Code가 자동으로 실행하는 명령
pip install openpyxl python-docx pandas
```

> ⚠️ **주의**: 첫 실행 시 라이브러리 설치에 시간이 걸릴 수 있습니다.

## 2. 엑셀/CSV 자동 처리

### 2-1. CSV 데이터 분석 — 자연어로 데이터 다루기

Claude Code에 CSV 파일을 주고 자연어로 요청하면, 파이썬 코드를 생성·실행합니다:

```
input/survey_results.csv를 분석해줘:
1. 전체 응답자 수
2. 학년별 응답 비율
3. 만족도 평균과 표준편차
4. 학년별 만족도 차이가 통계적으로 유의미한지 (t-test)
결과를 output/survey_analysis.md에 정리해줘.
```

Claude Code는 이 요청을 받아 pandas와 scipy를 활용한 분석 코드를 자동 작성·실행합니다.

### 2-2. 엑셀 파일 생성 — 서식까지 포함

```
다음 데이터로 엑셀 파일을 만들어줘:

시트1 "동아리 회비":
| 이름 | 월 | 금액 | 납부일 |
데이터: [...]

서식 요청:
- 제목 행: 파란 배경, 흰색 굵은 글씨
- 금액 열: 천단위 쉼표, 우측 정렬
- 합계 행 추가
- 시트2에 월별 합계 차트 추가

output/club_fees.xlsx로 저장해줘.
```

### 2-3. CSV 병합 및 정리

여러 CSV 파일을 하나로 합치거나, 지저분한 데이터를 정리하는 작업:

```
input/ 폴더의 CSV 파일 3개를 하나로 합쳐줘:
- survey_2023.csv
- survey_2024_spring.csv
- survey_2024_fall.csv

합칠 때:
- 중복 행 제거
- 빈 칸은 "미응답"으로 채우기
- 날짜 형식을 YYYY-MM-DD로 통일
- 이름 열의 공백 제거

output/merged_survey.csv로 저장해줘.
```

## 3. 워드(DOCX) 자동 생성

### 3-1. 보고서 자동 생성

Claude Code로 서식이 포함된 워드 문서를 생성할 수 있습니다:

```
다음 내용으로 보고서를 워드(docx) 문서로 만들어줘:

제목: [보고서 제목]
학과: [학과명]  |  학번: [학번]  |  이름: [이름]

서식:
- 제목: 16pt, 굵게, 가운데 정렬
- 본문: 11pt, 줄간격 1.5
- 페이지 번호 포함
- 목차 자동 생성

내용은 _workspace/report_draft.md를 참고해줘.
output/final_report.docx로 저장해줘.
```

### 3-2. 데이터 → 문서 파이프라인

데이터를 분석한 결과를 바로 문서로 만드는 워크플로우:

```
input/experiment_data.csv → 분석 → output/lab_report.docx

1. CSV에서 평균, 표준편차, 그래프 생성
2. 분석 결과를 해석하는 텍스트 작성
3. 표와 그래프를 포함한 실험 보고서 docx 생성
```

## 4. 파일 처리 에이전트 팀

### 4-1. 에이전트 팀 구성

```
.claude/
  agents/
    data-cleaner.md    ← 데이터 정제 전문가
    data-analyst.md    ← 데이터 분석 전문가
    report-formatter.md ← 문서 생성 전문가
```

**data-cleaner.md**:
```markdown
---
name: data-cleaner
description: "CSV/XLSX 데이터를 정제하고 표준화하는 에이전트"
---
# Data Cleaner — 데이터 정제 전문가

## 핵심 역할
- 원본 데이터의 오류, 결측치, 중복을 처리합니다.
- 날짜, 숫자, 텍스트 형식을 표준화합니다.
- 정제 전후 데이터 품질 리포트를 작성합니다.

## 작업 원칙
- 원본 파일은 절대 수정하지 않습니다 (input/ 폴더 보존).
- 정제된 데이터는 _workspace/에 저장합니다.
- 정제 과정을 _workspace/cleaning_log.md에 기록합니다.
- 결측치 처리 방법: 삭제가 아닌 명시적 표시 ("미응답", "N/A")

## 출력
- _workspace/cleaned_data.csv: 정제된 데이터
- _workspace/cleaning_log.md: 정제 과정 기록

## 협업
- 완료 후 data-analyst에게 정제된 데이터를 전달합니다.
```

**data-analyst.md**:
```markdown
---
name: data-analyst
description: "정제된 데이터를 분석하고 인사이트를 도출하는 에이전트"
---
# Data Analyst — 데이터 분석 전문가

## 핵심 역할
- 기술 통계(평균, 중앙값, 표준편차, 빈도)를 계산합니다.
- 데이터 시각화(차트, 그래프)를 생성합니다.
- 핵심 인사이트와 해석을 작성합니다.

## 작업 원칙
- 분석 코드를 _workspace/analysis_code.py에 저장합니다.
- 차트는 output/charts/ 폴더에 PNG로 저장합니다.
- 인사이트는 수치 근거와 함께 제시합니다.
- 상관관계와 인과관계를 구분합니다.
- 본문의 PMC 연구 경고 준수: **기초 통계(평균, 빈도, 교차분석)는 AI에 신뢰**, 복잡한 통계(다중회귀, 시계열)는 교차 검증 필요성을 명시합니다.

## 출력
- _workspace/analysis_results.md: 분석 결과
- output/charts/: 시각화 파일들

## 협업
- data-cleaner의 정제 데이터를 입력으로 사용합니다.
- report-formatter에게 분석 결과와 차트를 전달합니다.
```

#### Step 2. 데이터 분석 ⏱️ 10분

```
input/ai_survey.csv를 분석해줘:

기술 통계:
- AI 도구 사용 비율 (도구별)
- 학년별/전공별 AI 활용빈도
- 만족도 평균 (전체 + 도구별)
```

> **핵심 정리**: 본문에서는 ChatGPT에 수식을 생성받아 엑셀에 붙여넣고, Code Interpreter로 데이터를 분석하고, VBA로 서식을 자동화하는 **대화형 접근**을 배웠습니다.
