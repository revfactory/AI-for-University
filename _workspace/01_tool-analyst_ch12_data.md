### 1.2 ChatGPT Code Interpreter 심층 분석

**핵심 원리:**
- 사용자가 파일을 업로드하면, ChatGPT가 **자동으로 Python 코드 작성·실행**
- pandas로 데이터 처리, Matplotlib/Plotly로 시각화
- 사용자는 코드를 볼 필요 없이 자연어로 분석 요청

**지원 차트 유형:**
- 정적: 히스토그램, 산점도, 박스플롯, 히트맵, 영역차트, 레이더, 트리맵, 버블, 워터폴
- **인터랙티브 전환:** 정적 차트 생성 후 우측 상단 "Switch to interactive chart" 클릭

**데이터 분석 능력:**
| 분석 유형 | 능력 | 설명 |
|----------|------|------|
| 기술통계 | ★★★★★ | 평균/중앙값/표준편차/사분위수 자동 계산 |
| 데이터 정제 | ★★★★★ | 결측치 처리, 이상치 탐지, 데이터 타입 변환 |
| 시각화 | ★★★★★ | 10+ 차트 유형, 인터랙티브 전환 |
| 통계 검정 | ★★★★☆ | t-검정, ANOVA, 상관분석, 회귀분석 |
| 예측 모델링 | ★★★★☆ | 회귀, 시계열, 기본 ML 모델 |
| 결과 해석 | ★★★★★ | 비전공자 눈높이 설명 |

**장점:** 코딩 제로, 파일 업로드만으로 완전한 분석. 자동 코드 재시도.
**한계:** 대용량 파일(100MB+) 처리 느림, 세션 종료 시 데이터 사라짐

### 1.3 Claude Artifacts 데이터 분석 심층

**Artifacts 시각화 특징:**
- **React 기반 인터랙티브 컴포넌트** 생성
- 대시보드, 차트, 인터랙티브 테이블을 사이드 패널에서 실시간 프리뷰
- 코드 수정 없이 자연어로 디자인/데이터 변경

**인라인 시각화 (2026.03 베타):**
- 채팅 내에서 직접 차트/다이어그램 렌더링
- HTML + SVG + Chart.js 기반, 호버/클릭 인터랙션
- Artifacts(사이드패널)와 달리 대화 흐름 속 임시 시각화
- 모든 플랜에서 사용 가능

**Claude vs ChatGPT 데이터 분석 비교:**

| 항목 | ChatGPT Code Interpreter | Claude Artifacts |
|------|-------------------------|------------------|
| 분석 방식 | Python 자동 실행 (서버사이드) | JavaScript/React 기반 (클라이언트사이드) |
| 코드 실행 | ✅ 실제 Python 실행 | ⚠️ 프리뷰 환경 (제한적 실행) |
| 대용량 처리 | 100MB 제한 | **1M 토큰** (텍스트 기반) |
| 시각화 | Matplotlib/Plotly (정적→인터랙티브) | **React 컴포넌트** (네이티브 인터랙티브) |
| 통계 엄밀성 | ★★★★★ (scipy/statsmodels) | ★★★★☆ (JavaScript 라이브러리) |
| 결과 재사용 | ⚠️ 세션 종료 시 삭제 | ✅ Artifacts로 저장·공유 |
| 적합한 경우 | **정확한 통계 분석** | **인터랙티브 대시보드** |

## 2. 공공데이터 분석 실습

### 2.1 공공데이터 포털 안내

**data.go.kr (공공데이터포털):**
- 한국 정부 공공데이터 통합 포털
- **파일 데이터:** CSV/Excel 형태로 로그인 없이 다운로드 가능
- **Open API:** 회원가입 + 활용 신청 필요 (개발 계정: 일일 1,000건 / 운영 계정: 일일 100,000건)
- JSON/XML 형식 REST API 제공
- Python, JavaScript, Java, C#, PHP 등 다국어 샘플 코드 제공

**학생 친화 데이터셋 추천:**

| 데이터셋 | 출처 | 분석 예시 |
|---------|------|----------|
| 서울시 대중교통 이용량 | data.seoul.go.kr | 시간대별 승하차 패턴 분석 |
| 전국 대학교 현황 | data.go.kr | 지역별 대학 수, 학생 수 비교 |
| 기상 데이터 | data.kma.go.kr | 연도별 기온 변화 추이 |
| 소비자물가지수 | kostat.go.kr | 품목별 물가 변동 시각화 |
| 취업률 통계 | kess.kedi.re.kr | 전공별 취업률 비교 분석 |
| 교통사고 통계 | taas.koroad.or.kr | 지역별/시간대별 사고 패턴 |

### 2.2 공공데이터 분석 실습 프롬프트

**Step 1: 데이터 탐색**
```
이 CSV 파일을 분석해주세요.

먼저 다음을 알려주세요:
1. 전체 행/열 수와 각 컬럼명
2. 각 컬럼의 데이터 타입과 결측치 비율
3. 수치형 컬럼의 기초 통계 (평균, 중앙값, 최솟값, 최댓값)
4. 범주형 컬럼의 고유값 개수와 상위 5개 빈도
5. 이 데이터로 할 수 있는 분석 3가지를 제안해줘
```

**Step 2: 시각화 요청**
```
이 데이터로 다음 차트를 만들어줘:

1. 연도별 추이 — 꺾은선 그래프
2. 카테고리별 비교 — 수평 막대그래프 (상위 10개)
3. 분포 확인 — 히스토그램 + 박스플롯
4. 상관관계 — 산점도 (추세선 포함)

차트 조건:
- 한국어 제목과 축 레이블
- 색상은 파랑~초록 계열 (시인성 높게)
- 각 차트 아래 1~2줄 해석 코멘트 추가
```

**Step 3: 인사이트 도출**
```
지금까지 분석한 결과를 바탕으로:

1. 핵심 인사이트 5가지를 정리해줘
2. 각 인사이트에 근거 데이터(수치)를 포함
3. 대학생 리포트에 쓸 수 있는 "분석 결과" 문단을 작성해줘
4. 한계점과 추가 분석이 필요한 부분도 명시
```

### 2.3 프롬프트 진화 패턴

**🔴 Level 1 (초보):**
```
이 엑셀 파일 분석해줘
```

**🟡 Level 2 (중급):**
```
이 CSV 데이터의 트렌드를 분석하고 차트를 만들어줘
```

**🟢 Level 3 (고급):**
```
서울시 대중교통 데이터 (CSV 첨부)를 분석해주세요:
1. 시간대별(0~23시) 평균 승차 인원 → 꺾은선 그래프
2. 노선별 상위 20개 이용량 → 수평 막대그래프
3. 주중 vs 주말 이용 패턴 차이 → 그룹화된 막대그래프
4. 월별 이용량 추이 → 영역 차트
5. 환승역 네트워크 분석 → 히트맵
모든 차트: 한국어 레이블, 인터랙티브, 인사이트 코멘트 포함
최종 요약: 핵심 발견 5가지 + 정책 제안 2가지
```

## 3. Gemini + Google Sheets 데이터 분석

### 3.1 2026년 최신 기능

**SpreadsheetBench SOTA (2026.03.10):**
- 912개 실제 포럼 기반 스프레드시트 작업에서 **70.48% 성공률** 달성
- 복잡한 멀티테이블 레이아웃, 실제 사용자 문제 해결
- Google 공식 리더보드 제출 및 검증 완료

**핵심 기능:**

1. **자연어 스프레드시트 생성:**
   - "시카고 이사 준비 — 방별 짐 체크리스트, 유틸리티 연락처, 이사업체 견적 비교표를 만들어줘"
   - Gmail, Drive의 개인 데이터를 참조하여 자동 채움

2. **Fill with Gemini:**
   - 커스텀 텍스트 생성, 데이터 분류, 요약
   - Google 검색 실시간 정보 접근
   - 예: "이 회사명 옆에 업종과 대표이사를 채워줘"

3. **멀티 테이블 분석 (2025.10~):**
   - 하나의 탭 내 여러 테이블을 이해하고 교차 분석
   - 학생 예: 중간고사 테이블 + 기말고사 테이블을 합쳐 종합 성적 분석

4. **자연어 데이터 분석:**
   - "이 데이터의 트렌드를 분석해줘" → 자동 차트 + 인사이트
   - "상위 10%를 강조 표시해줘" → 조건부 서식 자동 적용

**접근:** Google AI Ultra($19.99/월) / Pro 구독자 (영어, 글로벌 베타)

## 4. Julius AI 심층 분석

- **자연어 데이터 분석:** "이 데이터에서 이상치를 찾아줘" → 자동 분석
- **자동 차트 추천:** 데이터 특성에 맞는 최적 시각화 자동 선택

## 5. 노코드 데이터 분석 워크플로우

### 5.1 전체 파이프라인

```
[데이터 획득]
  → 공공데이터포털(data.go.kr) CSV 다운로드
  → 또는 직접 수집한 설문/실험 데이터

[1단계: 데이터 탐색 — ChatGPT Code Interpreter]
  → 파일 업로드 → "이 데이터를 탐색해줘"
  → 구조 파악, 결측치 확인, 기초 통계

[2단계: 데이터 정제 — ChatGPT/Claude]
  → "결측치를 평균값으로 대체하고, 이상치를 제거해줘"
  → 정제된 데이터 다운로드

[3단계: 시각화 — Claude Artifacts / ChatGPT]
  → "인터랙티브 대시보드를 만들어줘" (Claude)
  → 또는 "차트 5개를 만들어줘" (ChatGPT)

[4단계: 인사이트 도출 — ChatGPT/Claude]
  → "핵심 발견 5가지와 시사점을 정리해줘"

[5단계: 보고서 통합]
  → 차트 이미지 다운로드 → Word/PPT에 삽입
  → AI가 작성한 분석 문장을 보고서에 활용
```

### 5.2 도구별 역할 분담

| 단계 | 최적 도구 | 이유 |
|------|----------|------|
| 데이터 탐색 | ChatGPT Code Interpreter | Python 자동 실행, 가장 빠른 탐색 |
| 데이터 정제 | ChatGPT Code Interpreter | pandas 기반 정확한 처리 |
| 정적 시각화 | ChatGPT Code Interpreter | Matplotlib/Plotly 고품질 차트 |
| 인터랙티브 시각화 | Claude Artifacts | React 기반 대시보드, 호버/클릭 |
| 대용량 텍스트 분석 | Claude (1M 토큰) | 긴 보고서, 대규모 텍스트 데이터 |
| 스프레드시트 내 분석 | Gemini + Google Sheets | Sheets를 떠나지 않고 분석 |
| 빠른 탐색 분석 | Julius AI | 자동 차트 추천, 리포트 생성 |

## 7. 상황별 도구 선택 가이드

### 7.1 분석 목적별 추천

| 상황 | 추천 도구 | 이유 |
|------|----------|------|
| **과제 보고서용 간단 분석** | ChatGPT (Free) | 무료, 기본 분석+차트 |
| **인터랙티브 대시보드 필요** | Claude Artifacts | React 기반 대시보드 |
| **Google Sheets에서 바로 분석** | Gemini + Sheets | 도구 전환 없이 분석 |
| **대용량 데이터 (10만행+)** | ChatGPT Plus Code Interpreter | Python 서버 실행 |
| **텍스트 데이터 분석** | Claude (1M 토큰) | 긴 텍스트 처리 최강 |
| **예측 모델링** | Julius AI / ChatGPT | 시계열, 회귀 예측 |
| **경진대회 참가** | ChatGPT + Claude 조합 | 분석(ChatGPT) + 시각화(Claude) |
| **졸업논문 데이터 분석** | Jamovi + ChatGPT | 엄밀 통계 + 해석 보조 |

### 7.2 "파이썬 없이" 핵심 메시지

| 과거 (코딩 필요) | 현재 (AI 활용) |
|-----------------|---------------|
| Python 설치 + 라이브러리 세팅 | 파일 업로드 한 번 |
| pandas 문법 학습 | "이 데이터 분석해줘" 한 줄 |
| Matplotlib 차트 코드 작성 | "막대그래프 만들어줘" |
| 에러 디버깅 수시간 | AI가 자동 코드 수정·재실행 |
| 통계 공식 직접 구현 | "t-검정 해줘" → 결과 + 해석 |

> **핵심:** 코딩 능력이 아닌 **질문 능력(프롬프트)**이 데이터 분석의 핵심 역량이 되었다.
