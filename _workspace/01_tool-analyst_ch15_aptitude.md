- 평가 항목: 논리력, 집중력, 기억력, 창의력, 문제해결력 등

**연습 방법:**
1. **잡다 튜토리얼 (무료):** jobda.acca.ai/tutorial — 실제 검사와 동일한 형태 연습

3. **YouTube 연습 영상:** 게임 유형별 공략법
4. **반복 연습:** 반응 속도가 핵심이므로 게임 유형에 익숙해지는 것이 중요

**결과 활용:**
- 역량 유형과 강약점 분석 결과표 제공
- 면접 가이드 연동 — 역량 결과에 맞는 면접 준비 조언

## 2. NCS 필기시험 AI 활용

### 2.1 NCS 영역별 AI 활용법

**NCS 5대 영역 (공공기관 통합채용):**

| 영역 | 문항 수 | AI 활용법 |
|------|---------|----------|
| **의사소통능력** | 10문항 | Claude — 긴 지문 독해, 요약, 논리적 글쓰기 연습 |
| **수리능력** | 10문항 | ChatGPT — 자료해석, 응용계산 문제 생성 + 풀이 |
| **문제해결능력** | 10문항 | ChatGPT — 논리퀴즈, 상황판단 문제 생성 |
| **자원관리능력** | 10문항 | ChatGPT — 시간/예산/인력 배분 문제 |
| **조직이해능력** | 10문항 | Perplexity — 최신 조직/경영 시사 반영 문제 |

### 2.2 NCS 문제 생성 프롬프트

**수리능력:**
```
NCS 수리능력 영역의 자료해석 문제를 5개 만들어줘.

조건:
1. 난이도: 중~상 (공공기관 필기시험 수준)
2. 표나 그래프 데이터를 포함한 문제
3. 5지선다 객관식
4. 각 문제에 풀이 과정을 단계별로 설명
5. 시간 목표: 문제당 2분
6. 함정 선지(오답 유인)를 포함하고, 왜 틀리기 쉬운지 설명

출력 형식:
- 문제 → 선지 → 정답 → 풀이 → 오답 분석
```

**문제해결능력:**
```
NCS 문제해결능력 영역의 논리 추론 문제를 5개 만들어줘.

유형:
- 명제 논리 (참/거짓 판단)
- 배열/순서 문제 (조건 기반 배치)
- 상황판단 (최적 의사결정)

각 문제에:
1. 5지선다 선지
2. 정답과 풀이
3. 이 유형의 빠른 풀이 전략(팁)
```

### 2.3 대기업 인적성 AI 대비

| 시험 | 기업 | AI 활용 전략 |
|------|------|-----------|
| **GSAT** | 삼성 | ChatGPT로 수리·추리 문제 대량 생성 + 시간 관리 연습 |
| **HMAT** | 현대차 | 상황판단 시나리오 생성 + 논리 추론 연습 |
| **LG Way Fit** | LG | 조직적합성 문제 → ChatGPT로 가치관 정리 |
| **SK SKCT** | SK | 인지역량 + 심층역량 → 논리·수리 문제 + 가치관 |
| **CJ CAT** | CJ | 상식·시사 → Perplexity로 최신 뉴스 정리 |

## 3. AI 오답 노트 자동화

### 3.1 오답 분석 프롬프트

```
다음은 내가 틀린 NCS 수리능력 문제들입니다:

[틀린 문제 + 내가 고른 답 + 정답 붙여넣기]

각 문제에 대해:
1. 내가 틀린 이유 분석 (계산 실수? 개념 오해? 시간 부족?)
2. 이 유형의 핵심 풀이 공식/전략
3. 유사 문제 2개씩 추가 생성 (같은 유형, 다른 숫자)
4. 내 약점 패턴 종합 분석
5. 약점 보완을 위한 1주일 학습 계획

출력:
- 약점 유형별 분류표
- 우선순위 학습 리스트 (가장 자주 틀리는 유형부터)
```

### 3.2 약점 진단 + 맞춤 학습 계획

```
나는 NCS 필기시험을 준비하고 있습니다.

다음은 최근 모의시험 결과입니다:
- 의사소통: 7/10
- 수리능력: 5/10
- 문제해결: 6/10
- 자원관리: 8/10
- 조직이해: 4/10

시험까지 남은 기간: [D-일수]

분석해주세요:
1. 과목별 약점 분석
2. 우선순위 학습 순서 (효율 극대화)
3. 일별 학습 계획표
4. 각 과목별 핵심 공략 포인트
5. 매일 연습할 문제 수 제안
```

1. 시험에서 측정하는 핵심 역량은?
2. 프롬프트 작성의 핵심 원칙 5가지

## 5. 프롬프트 진화 패턴

**🔴 Level 1 (초보):**
```
GSAT 문제 만들어줘
```

**🟡 Level 2 (중급):**
```
삼성 GSAT 수리 영역 자료해석 문제 10개와 풀이를 만들어줘
```

**🟢 Level 3 (고급):**
```
삼성 GSAT 수리영역 대비:
1. 자료해석 5문제 (표+그래프 포함, 난이도 상)
2. 응용계산 5문제 (속도/농도/확률)
3. 각 문제: 5지선다 + 풀이 + 오답 유인 분석 + 풀이 시간 목표
4. 내가 틀린 문제: [이전 오답 리스트] → 약점 유형 재출제
5. 시간 관리 전략 (문제당 목표 시간)
출력 마지막에: 이번 세트의 난이도 평가 + 추가 연습 추천
```
