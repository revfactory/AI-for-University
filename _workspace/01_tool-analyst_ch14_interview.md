### 1.2 ChatGPT 모의면접 심층 분석

**모의면접 활용법:**

1. **예상 질문 생성:**
   - 기업별/직무별 맞춤 질문 생성
   - 인성 면접 + 직무 면접 + PT 면접 대비

2. **STAR 기법 답변 코칭:**
   - Situation(상황) → Task(과제) → Action(행동) → Result(결과)
   - 경험을 STAR 구조로 재구성 + 수치화

3. **모의면접 시뮬레이션:**
   - 면접관 역할 수행 → 답변 → 피드백 → 개선
   - 후속질문으로 압박 면접 연습

4. **Voice Mode 활용:**
   - 실제 말하기 연습 (텍스트가 아닌 음성 대화)
   - 답변 시간, 필러워드 체크

**ChatGPT가 효과적인 이유:**
- 81%의 구직자가 AI를 활용한 STAR 기법 연습 후 면접 자신감 향상 (Parakeet AI 2026 조사)
- 3~5회 모의면접 연습 후 유의미한 답변 품질 개선

## 2. 기업 분석 도구

### 2.1 Perplexity로 기업 분석

**Perplexity가 기업 분석에 최적인 이유:**
- 실시간 웹 검색 + 출처 표시 → 최신 기업 정보 확보
- 경쟁사 비교, 재무 데이터, 최근 뉴스 종합 분석
- 면접 질문과 연결되는 기업 전략/이슈 파악

**기업 분석 프롬프트:**
```
[기업명]에 대해 면접 준비용 분석을 해주세요:

1. 기업 개요: 매출, 직원 수, 핵심 사업
2. 최근 1년 주요 뉴스/이슈 5가지
3. 경쟁사 대비 강점/약점
4. 산업 트렌드와 기업의 대응 전략
5. 인재상 및 핵심 가치
6. 이 기업 면접에서 나올 수 있는 질문 5가지
7. "왜 이 회사에 지원했는가"에 대한 답변 소재 3가지

출처를 반드시 포함해주세요.
```

## 3. 면접 유형별 AI 활용 전략

### 3.1 인성 면접

```
나는 [기업명] [직무]에 지원한 [전공] 학생입니다.

인성 면접 예상 질문 10가지와 각 질문에 대한 모범 답변을 작성해주세요.

조건:
1. STAR 기법 적용
2. 나의 경험: [핵심 경험 3가지 나열]
3. 기업 인재상: [인재상 입력]
4. 답변 길이: 1분 30초 분량 (약 300자)
5. 각 답변에 "면접관이 할 수 있는 후속 질문"도 추가

질문 유형 포함:
- 지원 동기
- 성격 장단점
- 갈등 해결 경험
- 팀워크 경험
- 실패 극복 경험
```

### 3.2 직무 면접 / 기술 면접

```
[직무]의 기술 면접을 준비하고 있습니다.

1. 해당 직무의 핵심 역량 5가지
2. 각 역량별 예상 질문 2개씩 (총 10개)
3. 각 질문에 대한 모범 답변 (기술적 정확성 + 실제 경험 연결)
4. 면접관이 기대하는 포인트 (채점 기준)

내 기술 스택: [사용 가능한 도구/언어/프레임워크]
관련 프로젝트: [프로젝트 경험 요약]
```

### 3.3 PT 면접

```
[기업명] PT 면접을 준비합니다.

예상 주제 3가지를 제시하고, 각 주제에 대해:
1. 10분 발표용 슬라이드 구조 (6~8장)
2. 각 슬라이드의 핵심 메시지
3. 예상 반론 및 대응 논리
4. 데이터/수치 근거 포함

산업: [산업 분야]
직무: [직무명]
최근 이슈: [관련 이슈]
```

### 3.4 AI 역량검사 대비 (잡다 ACCA)

```
AI 역량검사(ACCA)에 대해 알려주세요:

1. 검사 구성: 어떤 게임/과제가 포함되는지
2. 평가 항목: 논리력, 집중력, 기억력, 창의력 등
3. 유형별 연습 전략
4. 연습 가능한 앱/사이트
5. 고득점 전략

750+ 기업이 활용하는 잡다(JOBDA) 역량검사 기준으로 설명해줘.
```

## 4. 면접 준비 워크플로우

### 4.1 전체 파이프라인

```
[D-14] 기업 분석 (Perplexity)
  → 기업 개요, 최근 뉴스, 인재상, 산업 트렌드

[D-10] 예상 질문 생성 (ChatGPT)
  → 인성 10문 + 직무 10문 + PT 주제 3개

[D-7] 답변 작성 (ChatGPT + Claude)
  → STAR 구조 답변 초안 → Claude로 논리 보강

[D-5] 모의면접 연습 (ChatGPT Voice / Google Interview Warmup)
  → 음성 연습, 시간 체크, 필러워드 줄이기

[D-3] 피드백 반영 및 수정
  → "내 답변을 더 구체적으로/간결하게 수정해줘"

[D-1] 최종 정리
  → 핵심 답변 요약 카드 생성 (ChatGPT)
  → 면접장 질문 리스트 ("마지막으로 질문 있으시나요?")
```

### 4.2 프롬프트 진화 패턴

**🔴 Level 1 (초보):**
```
면접 질문 알려줘
```

**🟡 Level 2 (중급):**
```
삼성전자 SW직군 면접 예상 질문 10개와 답변을 만들어줘
```

**🟢 Level 3 (고급):**
```
삼성전자 DX부문 SW직군 면접 준비:
■ 인성질문 5개 (인재상 "함께가는 인재" 연결, STAR기법)
■ 기술질문 5개 (React/Node.js 기반, 프로젝트 경험 연결)
■ 각 답변 1분30초 분량 + 예상 후속질문 + 대응 전략
■ PT면접 주제 2개 (DX 산업 트렌드 기반, 슬라이드 구조 포함)
■ 역질문 3개 (인상적이면서 진정성 있는)
내 경험: [인턴/프로젝트/수상 경력 상세]
```
