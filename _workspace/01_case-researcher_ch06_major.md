## 1. 공학/컴퓨터과학

### 사례 1-1: GitHub Copilot이 학생의 프로그래밍에 미치는 효과 — ACM 연구 (✅ A급)

**출처:** "The Effects of GitHub Copilot on Computing Students' Programming Effectiveness, Efficiency, and Processes in Brownfield Coding Tasks", ACM ICER 2025 / arXiv
- ACM: https://dl.acm.org/doi/10.1145/3702652.3744219
- arXiv: https://arxiv.org/html/2506.10051v1

**핵심 데이터:**
- Brownfield(기존 코드 기반) 코딩 작업에서 Copilot 사용 학생 평가
- Copilot 사용 학생이 **유의미하게 더 효율적으로** 작업 완료
- **세분화된 수준에서 Copilot 제안을 채택한 학생이 통째로 채택한 학생보다 더 성공적**
- 코드 이해 기능(Copilot Chat)에 대한 신뢰도 > 코드 생성 기능 신뢰도
- 상위 학년 소프트웨어 공학 수업에서 실시

**교재 활용:** "AI 코딩 도구를 잘 쓰는 법 — 통째 복사가 아니라 조각 단위로"

**출처:** "Students' Use of GitHub Copilot for Working with Large Code Bases", ACM Technical Symposium on CS Education V.1 (2025)
- URL: https://dl.acm.org/doi/10.1145/3641554.3701800

**핵심 데이터:**
- 상위 학년 소프트웨어 공학 수업에서 학생들에게 Copilot 기능(Chat 포함) 교육 후 대규모 코드베이스에 기능 추가 과제 수행
- 학생들이 코드 생성보다 **코드 이해 기능**을 더 신뢰
- AI 도구를 통해 대규모 코드 프로젝트에 빠르게 적응 가능
