### 1.4 Jamovi 심층 분석 (SPSS 무료 대안)

**핵심 강점:**
- **완전 무료** 오픈소스, Windows/macOS/Linux/ChromeOS 지원
- **SPSS와 유사한 GUI:** 포인트&클릭으로 통계 분석 — 코딩 불필요
- **Excel/CSV 직접 로드:** JASP는 CSV만 지원하나, Jamovi는 Excel도 지원
- **R 기반:** 내부적으로 R 엔진 사용, 확장 모듈로 고급 분석 가능
- **교육용 추천:** 학부 수준 통계 교육에 SPSS보다 적합하다는 연구 결과

**Jamovi vs JASP 비교:**

| 항목 | Jamovi | JASP |
|------|--------|------|
| 가격 | 무료 | 무료 |
| UI 특징 | 스프레드시트형, 실용적 | 깔끔하고 직관적 |
| 파일 지원 | Excel + CSV | CSV 위주 |
| 통계 강점 | 빈도주의 통계, 혼합효과 모형 | **베이지안 통계** 최강 |
| APA 출력 | 모듈 필요 | **자동 APA 형식** |
| 확장성 | 커뮤니티 모듈 | 제한적 |
| 적합 학과 | 사회과학 전반, 경영학 | 심리학, 교육학 |
| 추천 | **학부 범용 1순위** | 베이지안 분석 필요 시 |

## 2. AI로 설문 데이터 분석하기

### 2.1 설문 분석 워크플로우

```
Step 1. 설문 수집 (Google Forms / 네이버 폼)
  → 응답 데이터를 CSV/Excel로 다운로드

Step 2. 데이터 정리 (ChatGPT/Claude)
  → "이 CSV 파일의 결측치를 확인하고, 이상치를 표시해줘"
  → 리커트 척도 코딩 확인 (1=전혀 아니다 ~ 5=매우 그렇다)

Step 3. 기초 통계 분석 (ChatGPT Code Interpreter)
  → 빈도분석, 기술통계(평균/표준편차), 교차분석
  → 차트 자동 생성 (막대/원형/박스플롯)

Step 4. 심화 분석 (Jamovi / ChatGPT)
  → t-검정, ANOVA, 상관분석, 회귀분석
  → p-value 해석 및 통계적 유의미성 판단

Step 5. 보고서 작성 (Claude Artifacts)
  → 인터랙티브 차트로 결과 시각화
  → 분석 결과 요약 및 시사점 도출
```

| 기술통계 | ★★★★★ | ★★★★★ | ★★★★☆ | ★★★★★ | ★★★★★ |

### 2.3 설문 분석 S급 프롬프트

**기초 통계 분석:**
```
다음은 대학생 학습 만족도 설문 결과입니다 (CSV 파일 첨부).

이 데이터를 분석해주세요:

1. 기초 통계
   - 응답자 인구통계 분포 (학년, 성별, 전공계열)
   - 각 문항별 평균, 표준편차, 빈도 분포
   - 리커트 5점 척도 문항은 막대그래프로 시각화

2. 교차분석
   - 학년별 만족도 차이 (χ² 검정)
   - 성별에 따른 AI 활용도 차이

3. 상관분석
   - 주요 변수 간 상관계수 행렬 (히트맵으로 시각화)
   - 유의미한 상관관계 해석

4. 결과 요약
   - 핵심 발견 3~5가지
   - 한계점 및 후속 연구 제안

모든 통계 결과에 p-value를 포함하고,
비전공자도 이해할 수 있게 쉽게 설명해주세요.
```

**교차분석 심화 프롬프트:**
```
첨부된 설문 데이터에서 다음 교차분석을 수행해주세요:

1. 독립변수: 학년 (1~4학년)
2. 종속변수: AI 활용 빈도 (매일/주3~4회/주1~2회/거의 안함)

분석 내용:
- 교차표 (빈도 + 기대빈도 + 잔차)
- χ²(카이제곱) 검정 결과 (χ² 값, df, p-value)
- 크래머의 V (효과 크기)
- 그룹화된 막대그래프

결과 해석을 보고서에 바로 쓸 수 있는 문장으로 작성해줘.
예시: "학년에 따른 AI 활용 빈도의 차이는 통계적으로 유의미하였다 (χ²=12.34, df=9, p=.031)."
```

### 2.4 프롬프트 진화 패턴

**🔴 Level 1 (초보):**
```
설문 결과를 분석해줘
```

**🟡 Level 2 (중급):**
```
이 설문 데이터의 기술통계와 교차분석을 해주고 차트를 만들어줘
```

**🟢 Level 3 (고급):**
```
첨부된 설문 CSV를 분석해주세요:
1. 인구통계 빈도분석 → 파이차트
2. 리커트 문항 기술통계 → 평균 비교 막대그래프
3. 학년×만족도 교차분석 (χ², Cramér's V)
4. 주요 변수 상관분석 → 히트맵
5. 회귀분석 (만족도 예측 모형)
6. 결과를 APA 7th 형식으로 보고서 문장 작성
모든 차트는 한국어 레이블, p<.05 유의수준 적용
```

## 4. 기초 통계 개념 + AI 해석

### 4.1 대학생이 알아야 할 통계 개념 × AI 활용

| 통계 개념 | 설명 | AI 활용법 |
|----------|------|----------|
| **기술통계** (평균, 표준편차, 중앙값) | 데이터의 중심과 퍼짐 정도 | "이 데이터의 기술통계를 구하고 해석해줘" |
| **빈도분석** | 각 응답의 빈도와 비율 | "각 문항별 빈도표와 막대그래프를 만들어줘" |
| **교차분석 (χ² 검정)** | 두 범주형 변수의 관계 | "성별×만족도 교차표와 카이제곱 검정을 해줘" |
| **t-검정** | 두 그룹 평균 비교 | "남녀 간 점수 차이가 유의한지 t-검정해줘" |
| **ANOVA** | 세 그룹 이상 평균 비교 | "학년별(1~4) 만족도 차이를 ANOVA로 분석해줘" |
| **상관분석** | 두 연속변수의 관계 | "학습시간과 성적 간 상관계수를 구해줘" |
| **회귀분석** | 예측 모형 | "만족도를 예측하는 변수를 회귀분석으로 찾아줘" |
| **신뢰도 분석 (Cronbach's α)** | 설문 문항의 내적 일관성 | "이 척도의 신뢰도를 분석해줘" |

### 4.2 AI에게 통계 결과 해석 요청 프롬프트

```
다음 통계 분석 결과를 해석해주세요:

[분석 결과 붙여넣기]

해석 조건:
1. 통계 비전공 대학생도 이해할 수 있는 쉬운 말로
2. "~이므로, ~라고 할 수 있다" 형식의 보고서 문장으로
3. p-value의 의미를 일상적인 비유로 설명
4. 실제 의미 (실용적 유의성)도 함께 설명
5. 주의할 점이나 한계도 언급
```

## 6. 상황별 도구 선택 가이드
