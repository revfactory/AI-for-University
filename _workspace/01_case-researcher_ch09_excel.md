## 1. 학술 논문 채널

### 사례 1-1: ChatGPT의 데이터 분석 성능 평가 — PMC 연구 (✅ A급)

**출처:** PMC (PubMed Central) "Examination of ChatGPT's Performance as a Data Analysis Tool" (2024)
- URL: https://pmc.ncbi.nlm.nih.gov/articles/PMC11696938/

**핵심 데이터:**
- ChatGPT의 데이터 분석 도구로서의 성능을 체계적으로 평가한 학술 연구
- 요약, 차트 생성, 회귀분석, 상관분석 등 통계 작업에서의 정확도 측정
- CSV, Excel, PDF, 텍스트 형식 파일 업로드 → Python 기반 실행 환경에서 분석
- **결론:** 기본적인 데이터 탐색과 시각화에는 효과적이나, **복잡한 통계 분석에서는 한계** 존재
- 전문 통계 소프트웨어(R, SPSS) 대비 정확도 차이 발생 가능

**교재 활용:** "ChatGPT 데이터 분석, 어디까지 믿을 수 있나?" — 학술 검증 결과

## 5. AI 도구 공식 채널

### 사례 5-1: ChatGPT Advanced Data Analysis — 공식 기능 (📝 B급)

**출처:** Numerous.ai (2025), ClickUp (2025)
- URL: https://numerous.ai/blog/can-chatgpt-analyze-excel-data
- URL: https://clickup.com/blog/how-to-use-chatgpt-for-excel/

**핵심 기능:**
- **파일 업로드 분석**: CSV, Excel, PDF 직접 업로드 → 자동 분석
- **Python 실행 환경 (Code Interpreter)**: 서버에서 Python 코드 실행 → 시각화/통계 결과 즉시 확인
- **자연어 질의**: "이 데이터에서 매출 트렌드를 보여줘" → 차트 자동 생성
- **[tool-analyst 교차] 수식 생성 능력: ★★★★★** — 자연어→엑셀 수식 변환 정확도 최고
- **[tool-analyst 교차] VBA 매크로 생성: ★★★★★** — 코드 품질 최고 수준
- **가격:** ChatGPT Plus ($20/월) 이상에서 사용 가능
- **Claude와 비교:** ChatGPT는 Code Interpreter로 실행까지 가능, Claude는 1M 토큰으로 긴 데이터셋 분석에 강점
- **한계:** 10만 행 이상 대용량 처리 시 속도 저하, 복잡한 매크로는 디버깅 필요

## 7. 실패 사례 / Anti-Pattern

### 사례 7-1: ChatGPT Excel 분석의 한계 — 복잡한 통계에서의 오류 (✅ A급 — 학술 검증)

**출처:** 사례 1-1 교차검증 (PMC 연구)

**핵심 교훈:**
- ChatGPT의 Excel 데이터 분석: 기본 탐색/시각화에는 효과적
- **복잡한 통계 분석에서 한계:**
  - 다중회귀분석: 변수 선택 근거 불충분
  - 시계열 분석: 계절성 패턴 간과 가능
  - 대용량 데이터: 10만 행+ 시 처리 오류 발생
- "ChatGPT가 만들어준 분석 결과를 그대로 과제에 넣었다가 교수님께 지적받음"
- **해결:** 기본 탐색은 ChatGPT, 정밀 분석은 R/SPSS/Excel 고급 기능 활용

**교재 활용:** "ChatGPT 데이터 분석, 이것만은 직접 해라" — 핵심 Anti-Pattern

---

### 사례 7-2: Excel 수식 맹신의 함정 (💡 C급)

**출처:** 합리적 추론 기반 구성 사례

**시나리오:**
- 대학생 B가 설문조사 데이터 분석 과제에서 ChatGPT에 "VLOOKUP 수식 만들어줘" 요청
- ChatGPT가 정확한 수식을 생성했으나, B가 **데이터 범위를 잘못 지정**
- 결과: 100명 중 20명의 데이터가 누락된 채 분석 완료 → 교수 피드백에서 오류 발견
- **교훈:** AI가 만든 수식은 정확하더라도, **적용하는 인간이 데이터를 이해해야 함**
- "AI는 수식을 만들어주지만, 데이터의 맥락은 인간만 알 수 있다"

**교재 활용:** "AI 수식의 함정 — 수식이 맞아도 결과가 틀릴 수 있다"
