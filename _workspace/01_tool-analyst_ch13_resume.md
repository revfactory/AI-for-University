### 1.3 AI 자소서 작성 시 주의사항

**AI 탐지 이슈:**
- 기업들이 AI 작성 자소서 탐지 도구 도입 확대 추세
- **GPTZero:** AI 텍스트 탐지 정확도 ~70~80% (perplexity + burstiness 분석)
- **카피킬러 AI 생성 검사:** 한국 기업 일부 도입
- **대응 전략:** AI 초안 → 본인 경험 중심 재작성 → 탐지기 자가 검수 (3단계)

**안전한 AI 활용 원칙:**
1. AI는 **구조·표현·아이디어 참고** 수준으로 활용
2. **핵심 경험과 논리는 반드시 본인이 직접 정리**
3. AI가 생성한 문장을 그대로 복사하지 않고 **자기 언어로 재구성**
4. 최종본을 AI 탐지기로 자가 검수

| **Gamma** | Free(400크레딧) / $8/월~ | AI로 포트폴리오 페이지 60초 생성, 웹 퍼블리시 | ★★★★★ | 빠른 포트폴리오 필요 시 |

| **GitHub Pages** | 무료 | 코드 기반 포트폴리오, 기술 역량 증명 | Copilot으로 코드 생성 | 개발자, 데이터 |

### 3.2 Notion 포트폴리오 구축 가이드

```
Step 1. Notion에 포트폴리오 페이지 생성
  → 포트폴리오 템플릿 활용 (Notion 마켓플레이스에서 무료 다운로드)
  → 섹션: About Me / Projects / Skills / Contact

Step 2. 프로젝트 데이터베이스 구성
  → 각 프로젝트: 제목, 설명, 역할, 기간, 성과, 이미지
  → 갤러리 뷰로 시각적 배치

Step 3. Notion AI로 콘텐츠 작성
  → "이 프로젝트 설명을 전문적이면서 읽기 쉽게 다듬어줘"
  → "내 역량을 3줄로 요약해줘"

Step 4. 웹사이트로 변환 (선택)
  → Super.so / Simple.ink 연동 → 커스텀 도메인으로 배포
  → 또는 Notion 공유 링크 그대로 사용
```

## 4. 자소서 작성 프롬프트

### 4.1 기업별 맞춤 자소서 S급 프롬프트

```
나는 [전공/학년]이고, [기업명]의 [직무]에 지원하려고 합니다.

다음 정보를 바탕으로 자기소개서를 작성해주세요:

■ 지원 기업 정보:
- 기업명: [기업명]
- 직무: [지원 직무]
- 자소서 항목: [항목 1~4 입력]

■ 나의 경험:
- 전공: [전공명]
- 관련 경험: [인턴/프로젝트/동아리 등 구체적 경험]
- 성과: [수치화 가능한 성과]
- 강점: [핵심 역량 2~3가지]

■ 작성 조건:
1. 각 항목 500~700자
2. STAR 기법 적용 (상황→과제→행동→결과)
3. 지원 기업의 인재상과 연결
4. 구체적 수치/사례 포함
5. AI가 쓴 티 나지 않는 자연스러운 문체
6. 각 항목의 핵심 메시지를 한 줄로 먼저 제시

주의: 이 초안을 바탕으로 내가 직접 수정할 예정이므로,
구조와 논리 흐름에 집중해주세요.
```

### 4.2 자소서 첨삭 프롬프트

```
다음 자기소개서를 첨삭해주세요:

[자소서 전문 붙여넣기]

첨삭 기준:
1. 논리적 흐름 (두괄식 구조인지)
2. 구체성 (추상적 표현 → 구체적 사례/수치)
3. 차별화 (다른 지원자와 구별되는 포인트)
4. 기업 연결성 (지원 기업의 인재상과 매칭)
5. 문장력 (장문 분리, 불필요한 수식어 제거)
6. AI 탐지 가능성 (너무 AI스러운 표현 체크)

출력 형식:
- 항목별 [현재 수준: A/B/C] + [개선 포인트]
- 수정된 문장 (원문 → 수정안)
- 전체 자소서 강점 3가지 / 약점 3가지
```

### 4.3 프롬프트 진화 패턴

**🔴 Level 1 (초보):**
```
삼성전자 자소서 써줘
```

**🟡 Level 2 (중급):**
```
삼성전자 DX부문 소프트웨어 직군 자소서를 써줘.
나는 컴퓨터공학과 4학년이고 개발 인턴 경험이 있어.
```

**🟢 Level 3 (고급):**
```
삼성전자 DX부문 SW직군 자소서 4개 항목 작성:
■ 경험: React 프론트엔드 인턴(3개월), 팀 프로젝트 리더(성과: 로딩속도 40% 개선)
■ 인재상 연결: 삼성 "함께가는 인재" + 기술 혁신
■ 조건: 항목별 600자, STAR기법, 두괄식, 수치화
■ 톤: 자신감 있되 겸손하게, AI스러운 표현 배제
이 초안을 바탕으로 내가 직접 수정할 예정
```

## 5. 포트폴리오 구조 설계 프롬프트

### 5.1 전공별 포트폴리오 프롬프트

```
나는 [전공] [학년]이고, [직무]에 지원하려고 합니다.

포트폴리오 구조를 설계해주세요:

포함할 프로젝트:
1. [프로젝트명 + 간단 설명]
2. [프로젝트명 + 간단 설명]
3. [프로젝트명 + 간단 설명]

설계 조건:
1. 각 프로젝트별 STAR 구조로 서술
2. 역할과 기여도를 명확히 표시
3. 정량적 성과 (수치, Before→After)
4. 사용 도구/기술 스택 명시
5. 프로젝트 간 스토리라인 (성장 서사)
6. "왜 이 직무에 적합한가"가 드러나도록

추가:
- About Me 섹션 (3~4줄 자기소개)
- Skills 섹션 (하드스킬 + 소프트스킬)
- 전체 포트폴리오 컨셉/키워드 제안
```
